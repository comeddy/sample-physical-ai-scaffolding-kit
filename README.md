# Physical AI Scaffolding Kit

[Physical AI 개발 지원 프로그램 by AWS Japan](https://aws.amazon.com/jp/blogs/news/aws-japan-physical-ai-development-support-program/)에서 소개된 샘플 리포지토리입니다.

이 리포지토리에는 다음 샘플이 포함되어 있습니다.

## 환경 구축

로컬의 제한된 리소스 환경에서는 효율적이고 빠르게 트레이닝을 반복하는 것이 과제가 될 수 있습니다.
Amazon SageMaker HyperPod는 생성형 AI 모델 구축에 수반되는 차별화와 무관한 번거로운 작업을 제거하고, 풍부한 리소스를 활용하면서 효율적으로 모델을 구축할 수 있게 해줍니다.

1. [Amazon SageMaker HyperPod를 활용한 Slurm 클러스터 구축](/hyperpod/README.md)
